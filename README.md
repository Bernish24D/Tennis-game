Turtle Tennis

Turtle Tennis is a fast-paced, competitive game where two players control their respective turtles to prevent the ball from crossing their side of the court.

Language : Python
Library : Turtle

Controls

Player 1:
w: Move up
s: Move down
a: Move left
d: Move right

Player 2:
u: Move up
j: Move down
h: Move left
k: Move right
